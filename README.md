# Panic Console: 20s Overload

> 中文：心跳控制台：20秒超載

像素風、直播互動的多工切換競技遊戲。螢幕**每 20 秒強制切換**到另一個同時運行的小遊戲——你要同時顧好多個遊戲、互相用卡牌整對手，還要讓觀眾用聊天室指令決定你的命運。

**目前階段**：前期製作 / 設計中（尚無程式碼）。

## 已確定範圍（截至 2026-06-14）

| 項目 | 內容 |
|------|------|

| 平台 | Steam |
| 引擎 | Unity (C#) |

完整定案與待定項目見 [定案規格.md](定案規格.md)。

目前進行：**Phase 0 垂直切片** — 設計見 [垂直切片 design](docs/superpowers/specs/2026-06-14-垂直切片-design.md)。
