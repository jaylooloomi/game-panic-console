## A. Steam 兩台電腦連線（1v1）

### A.0 機制（已查證）
- **Steam Datagram Relay (SDR) + SteamNetworkingSockets**：透過 Valve 中繼骨幹做 P2P，**不用自架伺服器**、隱藏 IP、自動 NAT 穿透，對 Steam 遊戲免費。
- Unity 接法：**Facepunch.Steamworks**（`InitRelayNetworkAccess` → `CreateRelaySocket` 開房 / `ConnectRelay(steamId)` 加入）＋ **Steam Lobby**（邀請/配對）。

### A.1 已就緒的架構縫
- `Assets/Scripts/Core/Net/INetTransport.cs`：傳輸抽象（Host/Join/Send/Poll/事件）。
- `Assets/Scripts/Core/Net/LocalTransport.cs`：本機 loopback 實作（已測試）。
- **對戰邏輯（DownstairsSim / BomberSim）已是「輸入驅動」**：第二位玩家的動作（Move/PlaceBomb/方向）目前由本機鍵盤產生。連線版只要把「P2 的輸入」改成從 `INetTransport.OnReceive` 解析而來、並把「本機 P1 的輸入」`Send` 出去即可。

### A.2 待你做的步驟
1. **取得測試 App ID**：開發階段在專案根放 `steam_appid.txt`，內容 `480`（Spacewar 測試用，免上架）。
2. **加入 Facepunch.Steamworks**：下載 `Facepunch.Steamworks`（GitHub/NuGet）的 `Facepunch.Steamworks.Win64.dll` 與 `steam_api64.dll`，放進 `Assets/Plugins/`。
3. **初始化**：遊戲啟動時 `SteamClient.Init(480)`；關閉時 `SteamClient.Shutdown()`。
4. **實作 `SteamRelayTransport : INetTransport`**（範本見 A.3）。
5. **連線測試（需兩台機或兩個 Steam 帳號）**：兩端都登入 Steam、直接執行 build 的 `.exe`（非從 library），一端 Host、另一端用對方 SteamID `Join`。

### A.3 SteamRelayTransport 範本（填入後即可用）
```csharp
// Assets/Scripts/App/Net/SteamRelayTransport.cs（需 Facepunch.Steamworks）
using System;
using Steamworks;
using Steamworks.Data;
using PanicConsole.Core.Net;

public class SteamRelayTransport : INetTransport, IConnectionManager, ISocketManager
{
    public NetRole Role { get; private set; }
    public bool IsConnected { get; private set; }
    public event Action<byte[]> OnReceive;
    public event Action OnConnected;
    public event Action OnDisconnected;

    SocketManager _socket;       // host 用
    ConnectionManager _conn;     // client 用

    public void Host()
    {
        Role = NetRole.Host;
        SteamNetworkingUtils.InitRelayNetworkAccess();
        _socket = SteamNetworkingSockets.CreateRelaySocket<SocketManager>();
        _socket.Interface = this;
        // 把自己的 SteamId（SteamClient.SteamId）透過 Lobby/邀請傳給對方
    }

    public void Join(string steamIdStr)
    {
        Role = NetRole.Client;
        SteamNetworkingUtils.InitRelayNetworkAccess();
        var id = new SteamId { Value = ulong.Parse(steamIdStr) };
        _conn = SteamNetworkingSockets.ConnectRelay<ConnectionManager>(id);
        _conn.Interface = this;
    }

    public void Send(byte[] data)
    {
        if (Role == NetRole.Host) { foreach (var c in _socket.Connected) c.SendMessage(data); }
        else _conn?.Connection.SendMessage(data);
    }

    public void Poll() { _socket?.Receive(); _conn?.Receive(); }
    public void Close() { _socket?.Close(); _conn?.Connection.Close(); IsConnected = false; OnDisconnected?.Invoke(); }

    // ISocketManager / IConnectionManager 回呼：
    public void OnConnecting(Connection c, ConnectionInfo i) => c.Accept();
    public void OnConnected(Connection c, ConnectionInfo i) { IsConnected = true; OnConnected?.Invoke(); }
    public void OnDisconnected(Connection c, ConnectionInfo i) { IsConnected = false; OnDisconnected?.Invoke(); }
    public void OnMessage(Connection c, NetIdentity id, IntPtr data, int size, long _, long __, int ___)
    {
        var buf = new byte[size]; System.Runtime.InteropServices.Marshal.Copy(data, buf, 0, size);
        OnReceive?.Invoke(buf);
    }
    // ConnectionManager 端的 OnConnecting/OnConnected/OnDisconnected/OnMessage 簽章略有不同，依 Facepunch wiki 對應。
}
```
> 細節以 Facepunch.Steamworks wiki 的 `SteamNetworkingSockets.CreateRelaySocket` / `ConnectRelay` 為準。Lobby（`SteamMatchmaking.CreateLobbyAsync` / 邀請好友）用來交換 SteamId 與配對。

### A.4 同步策略（建議）
- 對戰是 1v1、低頻輸入 → **輸入同步**最簡單：每端把自己的離散輸入事件（如 `DOWNSTAIRS_MOVE_L`、`BOMB_PLACE`）`Send`，兩端各自以相同邏輯推進。下樓梯/爆爆王皆為決定性步進，適合此法（注意亂數種子要一致：開房端決定種子並傳給對方）。
- 進階可改狀態同步（host 權威）以防分歧。

## B. Twitch 觀眾互動

### B.0 機制
- Twitch 聊天室走 **IRC 協定**（`irc.chat.twitch.tv:6667`，匿名讀取可用 `justinfan` 帳號免 OAuth）。觀眾打 `!ghost`、`!invert` 等指令 → 解析 → 觸發遊戲效果（對應卡牌系統）。

### B.1 已就緒的縫
- 卡牌效果系統（`CardEngine` + `SliceGame` 的效果套用）已存在；觀眾指令只要對應到同一組效果即可。
- 建議新增 `IChatSource`（事件：`OnViewerCommand(string cmd, string user)`）抽象，本機用假指令注入器測試、正式用 `TwitchChatSource`。

### B.2 待你做的步驟（需 Twitch 帳號/頻道）
1. 用 TCP 連 `irc.chat.twitch.tv:6667`，送 `NICK justinfan12345` + `JOIN #你的頻道`（匿名讀取）。
2. 解析 `PRIVMSG`，取訊息文字，比對 `!ghost`/`!invert`/`!speed` 等。
3. 累計計量表（規格 §5.2：滿額才發動），觸發對應效果（女鬼閃現/方向反轉…）。
4. 實測需開著你的 Twitch 直播/頻道、由真實觀眾或小號在聊天室打指令。

### B.3 TwitchChatSource 範本（讀取匿名聊天）
```csharp
// 需在背景執行緒讀 socket，主執行緒派發事件
using System.Net.Sockets; using System.IO;
var tcp = new TcpClient("irc.chat.twitch.tv", 6667);
var s = new StreamReader(tcp.GetStream()); var w = new StreamWriter(tcp.GetStream()){ AutoFlush = true };
w.WriteLine("NICK justinfan12345"); w.WriteLine("JOIN #yourchannel");
string line;
while ((line = s.ReadLine()) != null) {
    if (line.StartsWith("PING")) { w.WriteLine("PONG :tmi.twitch.tv"); continue; }
    // ":user!... PRIVMSG #chan :!ghost" → 取 ':' 後訊息，比對指令 → enqueue OnViewerCommand
}
```
